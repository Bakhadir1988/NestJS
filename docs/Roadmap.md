# Habit Tracker API — Roadmap

## Phase 1 — MVP (Core Learning Phase)

Цель: изучить базовую и среднюю архитектуру NestJS.

### Функциональность

- Users
- Auth (JWT access)
- Habits
  - daily / weekly
- Habit completion (1 раз в день)
- Streak calculation
- Basic statistics

### Технические темы

- Modules
- Controllers
- Services
- DTO & Validation
- Pipes
- Guards
- Prisma / TypeORM
- Transactions

---

## Phase 2 — Advanced Backend

Цель: архитектурное мышление и масштабирование.

### Функциональность

- Categories
- Tags
- Goals (X раз за период)
- Flexible schedules
- Soft delete
- Pagination / filtering

### Технические темы

- Interceptors
- Caching
- Cron jobs
- Advanced queries
- Performance optimization

---

## Phase 3 — Production / Senior Level

Цель: enterprise-подход.

### Функциональность

- Reminders
- Achievements
- Audit log
- Rate limiting
- Webhooks

### Технические темы

- CQRS
- Event-driven architecture
- Observability
- Monitoring
- Security hardening
