# Habit Tracker API — Product Vision

## Краткое описание

Habit Tracker API — это backend-приложение для отслеживания привычек,
ориентированное на чистую архитектуру, масштабируемость и production-подход.

Проект создаётся как учебный, но максимально приближенный
к реальному коммерческому продукту.

Референс продукта: **Habitify**.

---

## Целевая аудитория

- Пользователи, желающие отслеживать привычки
- Мобильные и web-клиенты
- Один пользователь — несколько привычек

---

## Проблема, которую решает продукт

- Отсутствие системного подхода к формированию привычек
- Нет прозрачной статистики и streak-механики
- Нет гибких правил выполнения

---

## Цели продукта

- Позволить пользователю:
  - создавать привычки
  - отмечать выполнение
  - видеть streak и статистику
- Обеспечить:
  - безопасную аутентификацию
  - строгую валидацию данных
  - расширяемую архитектуру

---

## Нефункциональные требования

- Чёткая модульная архитектура (NestJS)
- Типобезопасность
- Валидация на уровне входных данных
- Подготовка к масштабированию
- Готовность к production

---

## Границы MVP

В MVP **НЕ входят**:

- Push-уведомления
- Подписки
- Социальные функции
- Achievements

Они будут добавлены на следующих этапах.
